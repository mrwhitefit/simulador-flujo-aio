# Simulador El Flujo AIO

Demo interactiva para el directo "6.000€ antes de verano" (26 mayo 2026).

Visualiza el flujo completo del agente AIO en 6 pasos: marca → anuncios → clientes potenciales → exclusiva → RIC → venta cerrada.

## Stack

- Vite + React + TypeScript
- Tailwind CSS
- Framer Motion
- Deploy en Vercel

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Atajos de teclado

- `→` o `Espacio` — siguiente paso
- `←` — paso anterior
- `0-6` — ir a paso específico
- `R` — reset
